Ares Autosport Syncthing Setup

Prerequisites

  • SyncTrayzor
  • Cory, AP or Scott Server Device Id (Ask a team member for this)

Setup

  1. Close iRacing
  2. Browse to your setups folder, typically this is located in My Documents/iRacing
  3. Rename the setups folder
  4. Create a new empty setups folder
  5. Install Synctrayzor
  6. Go to Actions -> Settings and set your device name. The device name should make easy to recognise for others on the team
  7. Go to Add Remote Device Add the device id for one of the team servers and mark it as an introducer
  8. Send a message to the team member that owns the server and ask them to accept your connection. The team member will then share the setups folder with you
  9. Find the prompt for the iRacing Setups share and select Add
  10. In the General tab select browse in Folder Path and browse to your setups folder
  11. In the File Versioning tab select Simple File Versioning and clean out after 7 days
  12. Select Save
  13. The teams setups will now sync to you, this may take a short moment
  14. Select the share and then select Edit
  15. Open the Ignore Patterns tab, active Add ignore patterns and past the following into the text field:
#include .stglobalignore

Post Setup Steps

At this point you should have synced the teams setups, your old setups still remain in the folder you renamed in step 3. You may now copy over any setups that you need (It's good courtesy to stick to the established folder structure).

If you are using per car controls, ensure to copy those config files back to the corresponding car folder. These files are unique to you and do not sync to other team members.
